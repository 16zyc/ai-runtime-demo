## 1. 产品概述

AI Runtime 驾驶舱 Demo —— 全国首个面向 OPC 群体的"产业级 AI 操作系统"核心治理仪表盘。让开发区管理者像看"导航仪"一样看产业——实时、精准、可干预。本 Demo 聚焦驾驶舱五大核心模块的交互展示，使用模拟数据呈现完整的数据可视化与治理闭环体验。

- 面向开发区管委会领导、经发局、税务局、人社局、招商局等政府治理角色
- 核心价值：从"经验驱动"到"数据驱动"的治理转型

## 2. 核心功能

### 2.1 用户角色

| 角色 | 核心权限 |
|------|----------|
| 管委会领导 | 查看全部驾驶舱模块，接收风险预警简报 |
| 经发局/招商局 | 查看产业全景、人才洞察、创新增长模块 |
| 税务局 | 查看税收监管预警模块 |
| 人社局 | 查看社保扩面分析模块 |

### 2.2 功能模块

1. **驾驶舱主页**：顶部导航 + 五大模块入口 + 实时核心指标卡
2. **产业全景视图**：园区 OPC 总览、行业分布、收入水平、增长趋势
3. **社保扩面分析**：未参保率、人群特征交叉分析、地理分布热力图
4. **税收监管预警**：风险等级分布、异常交易检测、风险线索列表
5. **人才结构洞察**：技能标签云、紧缺人才预警、流动分析、高价值人才
6. **风险预警与主动服务**：风险发现→处置→反馈闭环展示

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色科技风（#0a0e1a 深蓝黑底色），搭配青色（#00e5ff）和琥珀色（#ffab00）作为强调色
- **辅助色**：风险红（#ff3d71）、预警黄（#ffaa00）、安全蓝（#0095ff）、成功绿（#00e096）
- **按钮风格**：圆角微光按钮，hover 时发光效果
- **字体**：数字/指标使用 DIN Alternate / JetBrains Mono，正文使用 Noto Sans SC
- **布局风格**：左侧固定导航 + 顶部面包屑 + 主内容区卡片式布局
- **图标风格**：线性图标（lucide-react），统一 1.5px 描边
- **动效**：数字滚动动画、图表入场动画、卡片悬浮微动效、脉冲呼吸灯（风险预警）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 驾驶舱主页 | 顶部指标卡 | 深色卡片 + 大号数字 + 趋势箭头 + 微光边框 |
| 驾驶舱主页 | 模块导航 | 5 个模块卡片，图标 + 名称 + 简要描述，hover 发光 |
| 产业全景视图 | 行业分布 | 环形图，中心显示总数，各扇区不同色 |
| 产业全景视图 | 收入趋势 | 渐变面积折线图，12 个月数据 |
| 社保扩面分析 | 人群特征 | 分组柱状图，年龄/行业维度 |
| 社保扩面分析 | 地理分布 | 模拟热力地图（网格化展示） |
| 税收监管预警 | 风险等级 | 三色环形进度条 + 数字 |
| 税收监管预警 | 风险列表 | 深色表格，行左侧色条标识风险等级 |
| 人才结构洞察 | 技能标签云 | CSS 词云，字号大小映射需求量 |
| 人才结构洞察 | 流动分析 | 横向双向柱状图（流入/流出） |
| 风险预警 | 事件流 | 时间线 + 脉冲圆点 + 状态标签 |

### 4.3 响应式

- 桌面优先设计（1920×1080 指挥中心大屏为主要场景）
- 适配 PC 端日常浏览（1366+ 宽度）
- 移动端提供简化版领导简报视图
